# Сайт-приглашение на свидание 💗

Готовый мини-сайт в стиле видео: карточки, котики, сердечки, плавные анимации, выбор даты/времени/еды и отправка результата в Telegram.

## Что внутри

- `Next.js` приложение
- интерактивные шаги приглашения
- анимированные SVG-стикеры без внешних файлов
- выбор даты и времени
- выбор еды
- API endpoint `/api/date-request`
- Telegram push через Bot API
- адаптация под телефон

## Быстрый запуск локально

```bash
npm install
npm run dev
```

Открой:

```text
http://localhost:3000
```

## Настройка Telegram-пушей

1. В Telegram открой `@BotFather`.
2. Создай бота командой `/newbot`.
3. Скопируй токен бота.
4. Напиши любое сообщение своему новому боту, например `привет`.
5. Узнай свой `chat_id`. Самый простой способ: временно открыть в браузере:

```text
https://api.telegram.org/botТВОЙ_ТОКЕН/getUpdates
```

В ответе найди `chat` → `id`.

6. Создай файл `.env.local` в корне проекта:

```env
TELEGRAM_BOT_TOKEN=сюда_токен_бота
TELEGRAM_CHAT_ID=сюда_chat_id
INVITE_OWNER_NAME=твое_имя
```

7. Перезапусти dev-сервер:

```bash
npm run dev
```

Теперь после финального подтверждения тебе будет приходить сообщение в Telegram.

## Деплой на Vercel

1. Загрузи проект на GitHub.
2. Импортируй репозиторий в Vercel.
3. В настройках проекта Vercel добавь Environment Variables:

```env
TELEGRAM_BOT_TOKEN
TELEGRAM_CHAT_ID
INVITE_OWNER_NAME
```

4. Нажми Deploy.

Важно: токен бота хранится только на сервере, не на клиенте. Не вставляй токен в браузерный JavaScript.

## Где менять тексты и варианты еды

Основной файл:

```text
app/page.tsx
```

В начале файла есть массив:

```ts
const foodOptions = [...]
```

Там можно поменять варианты еды, эмодзи и подписи.

Тексты экранов тоже находятся в `app/page.tsx`.

## Где менять стили и анимации

Файл:

```text
app/globals.css
```

Там находятся цвета, карточка, кнопки, плавающие сердечки и анимации котиков.
